更新 Homebrew 

```
brew update
```

更新所有安装过的软件包

```
brew upgrade
```

更新指定的软件包

```
brew upgrade wget
```

查找软件包

```
brew search wget
```

安装软件包

```
brew install wget
```

卸载软件包

```
brew remove wget
```

列出已安装的软件包

```
brew list
```

查看软件包信息

```
brew info wget
```

列出软件包的依赖关系

```
brew deps wget
```

列出可以更新的软件包

```
brew outdated
```

查找软件安装位置

```
which brew
```
